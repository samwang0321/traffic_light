# Traffic Light Detection with Voice Guidance

## 1. 專題簡介

本專題以 YOLOv8 物件偵測模型為核心，建立一套即時交通號誌辨識系統。系統透過攝影機擷取即時影像，辨識畫面中的紅燈與綠燈狀態，並結合語音播放功能，提供視障者即時通行提示。

當系統偵測到綠燈時，會播放「可以通行」語音提示；當偵測到紅燈時，會播放「禁止通行」語音提示。透過影像辨識與語音輔助，本系統可作為智慧輔助行人通行、無障礙交通提示與邊緣 AI 應用之基礎實作。

---

## 2. 系統需求

本系統需要具備以下功能：

* 即時讀取攝影機影像
* 使用 YOLOv8 模型進行交通號誌辨識
* 判斷紅燈與綠燈狀態
* 顯示即時辨識結果與 bounding box
* 根據辨識結果播放對應語音
* 支援自訂音檔播放
* 可於本機端離線執行

## 5. 執行方式

### 5.1 執行交通號誌辨識與語音提示程式

```bash
python test.py
```

執行後，系統會開啟攝影機並進行即時辨識。

### 5.2 操作方式

* 偵測到綠燈：播放「可以通行」
* 偵測到紅燈：播放「禁止通行」
* 按下 `q` 鍵：結束程式

### 5.3 攝影機編號設定

程式中使用：

```python
cap = cv2.VideoCapture(1)
```

如果攝影機無法開啟，可改成：

```python
cap = cv2.VideoCapture(0)
```

## 7. 模型或演算法說明

本系統採用 YOLOv8 作為物件偵測模型。YOLOv8 屬於 one-stage object detection 架構，能夠在單次推論中完成物件位置與類別判斷，適合應用於即時影像辨識任務。

### 系統流程

```text
攝影機擷取影像
        ↓
YOLOv8 模型推論
        ↓
辨識紅燈 / 綠燈
        ↓
繪製辨識框
        ↓
判斷燈號狀態
        ↓
播放對應語音提示
```

### 語音提示邏輯

系統會根據偵測結果播放對應音檔：

| 偵測結果 | 播放語音 |
| ---- | ---- |
| 綠燈   | 可以通行 |
| 紅燈   | 禁止通行 |

為避免語音過度重複，程式中可設定播放間隔，例如每 3 秒或 5 秒播放一次。

### 辨識信心值設定

程式中目前設定信心值為：

```python
conf=0.6
```

若偵測過於敏感，可提高信心值，例如：

```python
conf=0.7
```

若偵測不到目標，可降低信心值，例如：

```python
conf=0.5
```

### 測試說明
